How to create a Github page

Creating a repository

  • Sign in to your github account or create a new account if you don't have an account.
  • Click on the new icon to create a new repository.
  • Enter your name and create a new repository.

Creating a file

  • Open Visual Studio code editor and create a new folder with a new file - index.html.
  • Add your image file in the same folder.
  • Start writing your steps to create a website.
  • You can further design your page using CSS.
  • Add live server extension to check the file in your local browser.

Uploading the file

  • Save your file and add it to the new repository that was created by clicking uploading an existing file.
  • Drag the index.html and image files.
  • Commit the changes.
  • You can access your page at username.github.io.
